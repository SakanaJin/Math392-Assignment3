Below is instructions for running the program with a virtual environment.

Windows:
1.) python -m venv .venv //create venv
2.) .\.venv\Scripts\activate.bat (command prompt) //activate venv
    .\.venv\Scripts\Activate.ps1 (powershell)
3.) pip install -r requirements.txt //installing requirements not needed for runs after the first
4.) python main.py //run program
5.) deactivate //deactivate venv after running 

Linux:
1.) python3 -m venv .venv 
2.) source ./.venv/bin/activate
3.) pip install -r requirements.txt
4.) python3 main.py 
5.) deactivate
